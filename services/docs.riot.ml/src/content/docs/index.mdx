Riot is an opinionated stack for building applications in OCaml. It brings
together a single CLI, a multicore actor runtime, a large standard library, and
an artifact-native package registry.

This site documents the stack itself:

- how to install and upgrade Riot
- how to work with the `riot` CLI day to day
- how package management and publishing work
- how the runtime, `std`, and the registry fit together

Generated package documentation does not live here. Package docs are reserved
for `docs.pkgs.ml`.

If you want the deeper design rationale behind the current stack, the
[RFDs section](/rfds/) mirrors Riot's Requests for Discussion alongside these
guides.
