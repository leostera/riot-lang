---
title: "Isolated Installs"
description: "When Riot should materialize package state in a way that avoids accidental ambient coupling."
---

Isolated installs matter when you want package materialization to be explicit and bounded instead of
bleeding across unrelated local state.

## Why it matters

- reproducibility gets better
- debugging gets easier
- cross-package contamination gets less likely

This is an advanced topic because most users should not have to think about it on day one. But once
you care about CI, publishing, or tight operational guarantees, isolated installs become important.
