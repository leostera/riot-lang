---
title: "riot add"
description: "Add a dependency to a package and let Riot update the package graph coherently."
---

`riot add` is the normal way to grow a package’s dependency surface.

```sh
riot add colors
```

## Why it matters

Adding a package should do more than edit a manifest line. It should participate in the same package
management model as the rest of the toolchain:

- dependency selection
- lockfile updates
- local materialization
- predictable future builds

Use `riot add` whenever the dependency should become part of the package’s declared contract.
