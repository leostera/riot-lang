---
title: "riot login"
description: "Authenticate with the registry so publish and package-author flows can proceed."
---

`riot login` is the start of an authenticated package-author workflow.

Use it when you need to:

- publish a package
- interact with registry operations that require identity
- move from local package work to ecosystem-facing package work

This is intentionally not front-and-center in the main Toolchain story. Most users need to build and
run packages before they need to authenticate as publishers.
