---
title: "riot install"
description: "Install runnable binaries and materialize package-provided tools."
---

`riot install` is the command for turning package-provided binaries into locally usable tools.

## Typical reasons to use it

- install a tool from the package ecosystem
- materialize a binary without manually wiring paths
- make a package-provided executable available for local use

## Why it belongs in the package manager

Installation is where package resolution becomes an operational machine concern. The user should not
have to think in terms of tarballs, ad hoc directories, or one-off shell scripts to get a package
binary into use.
