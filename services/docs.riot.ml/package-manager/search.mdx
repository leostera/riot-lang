title: "riot search"
description: "Discover packages before you add them to a workspace."
---

`riot search` is the browse step before the dependency step.

Use it when you know the capability you want but do not yet know the package name or version shape.

## Why this command matters

Good package ecosystems are discoverable. Search lowers the cost of staying inside the ecosystem
instead of reaching for ad hoc external notes or memory.

Once you find the package you want, move directly to [riot add](/package-manager/add).
