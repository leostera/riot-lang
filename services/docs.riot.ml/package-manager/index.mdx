Riot’s package manager is not a separate product bolted onto the toolchain. It is the part that turns
package workflows into something repeatable and publishable.

## What this tab covers

- adding and removing dependencies
- updating and searching packages
- installing runnable binaries
- publishing to the registry
- generating package docs
- understanding lockfiles and caches

## Two ways to read it

- use the `Core Commands` group if you want operational flows
- use `Advanced Topics` if you need to understand why the system behaves the way it does
