# Changelog

All notable changes to `suri-mailer` are documented here.

## 0.1.0 - 2026-05-10

### Added

- Added Suri-oriented message composition helpers.
- Added local mailbox delivery for development and tests.
- Added filesystem outbox delivery.
- Added delivery tests for validation and adapter behavior.

### Changed

- Reported invalid messages, filesystem failures, and adapter failures with structured error variants.

## 0.0.34 - 2026-05-10

### Added

- Added a mail delivery package for Suri applications.
