🎉 Static Files Middleware

Production-ready static file serving for Riot/Suri

✅ Features

  • Security - Path traversal protection, dotfile blocking
  • Performance - ETag and Last-Modified caching, 304 responses
  • MIME Types - Automatic detection for 30+ file types
  • Directory Browsing - Optional HTML listings
  • Custom Headers - Add security headers, CORS, etc.

🔒 Security Tests

These should all be blocked:

  • Path Traversal - Should return 403
  • Dotfile Access - Should return 403

💾 Caching Test

Open DevTools Network tab and refresh this page. You should see:

  • First load: 200 OK with ETag header
  • Refresh: 304 Not Modified (no content transferred!)
  • Hard refresh: 200 OK again

📝 Usage

let app = Middleware.[
  logger;
  static ~at:"/public" (Path.v "./public") ();
  router routes;
]
