# Docstring Review

1. pick a file from the list below
2. review its docstrings:
    * make sure its formatting is github flavoured markdown
    * make sure the text is succint and accurate
    * remove any unnecessary text or listings or tables that are already part of the generated docs (like table of modules)
3. commit with conventional commit only your work
4. remove the file from the list below
5. repeat
